Privacy Policy

Fear Filter Chrome Extension · Last updated: February 2026
Summary: Fear Filter does not collect, store, or transmit any personal data to its developers. Images from your browser are sent directly to Anthropic's API using your own API key. Nothing passes through our servers — because we don't have any.

What data is processed

When Fear Filter is active, images found on web pages you visit are analyzed to detect content matching your selected fears. This analysis is performed by sending image data directly from your browser to the Anthropic Claude API (api.anthropic.com).

What we collect

Nothing. Fear Filter has no backend servers, no analytics, no tracking, and no data collection of any kind. The extension developer never receives any data about you, your browsing, or the images you view.

Your API key

Your Anthropic API key is stored using Chrome's built-in chrome.storage.sync API, which keeps it on your local device (and optionally synced across your Chrome profile via your Google account). The key is only ever transmitted directly to api.anthropic.com — the official Anthropic API endpoint — as part of image analysis requests.

Image data

Images are temporarily fetched and sent to Anthropic's API for classification. Fear Filter does not store images, log URLs, or retain any image data. Anthropic's own privacy policy governs how they handle API requests; you can review it at anthropic.com/privacy.

Permissions explained

  • storage — to save your API key and fear preferences locally
  • activeTab / scripting — to scan images on the page you're viewing when you click "Scan Page"
  • host_permissions (<all_urls>) — to fetch images from any website for analysis, and to run the content script that blurs detected images
  • api.anthropic.com — to send images to the Anthropic API for classification

Third-party services

Fear Filter uses the Anthropic Claude API to analyze images. Your use of this extension is also subject to Anthropic's Terms of Service and Privacy Policy.

Children's privacy

Fear Filter is not directed at children under 13 and does not knowingly collect information from children.

Changes to this policy

Any updates to this policy will be reflected in the extension's Chrome Web Store listing with an updated date above.

Contact

Questions about this privacy policy? You can reach us through the Chrome Web Store support tab for this extension.
